Q-Chambers Law Firm Management System

A comprehensive desktop application for managing your law firm's operations

Client Database

Manage client information including names, contact details, and addresses. Easily add, update, and search through your client list.

Case Management

Track all your cases with detailed information including case titles, statuses, open and close dates, and associated clients. Keep organized case histories and descriptions.

Document Storage

Store and manage case-related documents. Upload files, categorize by type, and link them to specific cases for easy retrieval.

Billing System

Handle invoicing and payments. Create bills for cases, track payment status, and maintain financial records.

Appointment Scheduling

Schedule and manage appointments, meetings, and court dates. Set reminders and associate appointments with clients and cases.

Technology Stack

  • Language: Java 11
  • GUI Framework: JavaFX 17
  • Build Tool: Maven
  • Architecture: MVC (Model-View-Controller)

How to Run the Application

  1. Ensure Java 11 or higher is installed on your system
  2. Install Maven (if not already installed)
  3. Open a terminal in the project root directory
  4. Run: mvn clean compile
  5. Run: mvn javafx:run

The application will launch with a main menu providing access to all features.
